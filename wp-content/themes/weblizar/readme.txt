== Description ==
Weblizar is a powerful and versatile WordPress theme with pixel perfect design and outstanding functionality. It is by far the most advanced free WordPress theme available today with loads of unmatched customization options.

<a href="https://wordpress.org/themes/weblizar/">Discover more about the theme</a>.
